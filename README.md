# Automação de Cadastro de Produtos

Projeto em Python para automação de cadastro de produtos em sistema web utilizando PyAutoGUI.

## Tecnologias
- Python
- PyAutoGUI
- Pandas

## Funcionalidades
- Login automático no sistema
- Importação de produtos via CSV
- Cadastro automático de produtos

## Bibliotecas utilizadas

pip install pyautogui
pip install pandas

## Estrutura do projeto

main.py → script principal  
pegar_posicao.py → captura coordenadas do mouse  
produtos.csv → base de dados dos produtos

## Demonstração
Automação que preenche formulários automaticamente em sistema web.
